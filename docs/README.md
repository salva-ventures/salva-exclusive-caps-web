# Documentation

Documentación interna de **Salva Exclusive Caps** para operación, web, estructura técnica y criterios base del proyecto.

---

## Overview

Este directorio concentra únicamente la documentación que aporta valor real al proyecto.  
La intención no es inflar el repositorio, sino mantener una base clara para operar mejor, escalar con orden y facilitar mantenimiento técnico.

## Principles

- Documentar solo lo útil
- Evitar duplicidad con el código
- Mantener textos claros y accionables
- Actualizar cuando cambie algo relevante
- Priorizar orden, no volumen

## Maintenance Rules

### Qué sí debe entrar aquí
- decisiones importantes
- procesos repetibles
- criterios técnicos del proyecto
- estructura del producto digital
- documentación que ayude a operar sin improvisar

### Qué no debe entrar aquí
- texto inflado
- ideas sueltas sin contexto
- documentación duplicada
- cosas que ya se entienden mejor directamente en el código
- documentos abandonados o contradictorios
