# Privacidad

> Borrador interno inicial. Debe revisarse antes de publicarse como versión final en la web.

## 1. Datos que podrían recopilarse
- nombre
- número de teléfono
- dirección de entrega
- ciudad o zona de envío
- datos compartidos por redes sociales
- información básica relacionada con el pedido

## 2. Finalidad
La información se utiliza únicamente para:
- atender solicitudes
- confirmar disponibilidad
- procesar pedidos
- coordinar entregas o envíos
- dar seguimiento comercial básico
- mejorar la operación

## 3. Criterio interno
Solo se debe solicitar la información mínima necesaria para completar una venta o dar seguimiento a un pedido.

## 4. Protección
La información de clientes debe manejarse con criterio, acceso limitado y uso estrictamente relacionado con la operación del negocio.

## 5. Actualización o eliminación
Cuando corresponda, el cliente podrá solicitar corrección o eliminación de la información compartida a través de los canales oficiales.
